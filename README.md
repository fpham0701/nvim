### Installation
To install this config, ensure `neovim` >= 0.9.0. 
Then, do the following:
```bashrc
mkdir ~/.config/
cd ~/.config
git clone https://github.com/fpham0701/nvim.git 
nvim
```
This should then open up `nvim` and install all the packages/dependencies used in this config.
